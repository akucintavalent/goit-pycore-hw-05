# Homework Assignment #5

A collection of Python tasks demonstrating advanced Python concepts including caching, generators, and decorators.

## Tasks Overview

### Task 1: Caching Fibonacci (`task1_caching_fibonacci.py`)

Implements a Fibonacci number calculator with caching for efficient computation.

**Features:**
- Returns a closure function that calculates Fibonacci numbers
- Uses an internal cache dictionary to store previously computed results
- Input validation for non-negative integers
- Avoids redundant recursive calculations

**Usage:**
```python
fib = caching_fibonacci()
print(fib(10))  # Returns 55
print(fib(15))  # Returns 610
```

---

### Task 2: Generator Numbers (`task2_generator_numbers.py`)

Extracts numerical values from text and calculates their sum.

**Features:**
- `generator_numbers(text)`: Generator function that yields all floating-point numbers found in text
  - Handles integers and decimals
  - Uses regex word boundaries to match numbers at any position
- `sum_profit(text, func)`: Calculates the sum of numbers generated by a provided function

**Usage:**
```python
text = "100.3 2000 50.8"
result = sum_profit(text, generator_numbers)
print(result)  # Returns 2151.1
```

---

### Task 3: Input Error Handling (`task4_input_error.py`)

Contact management system with error handling decorator for command-line interface.

**Features:**
- `input_error`: Decorator that catches `ValueError`, `KeyError`, and `IndexError` exceptions
- `parse_input(input)`: Parses user input into command and arguments
- `add_contact(args, contacts)`: Adds a new contact with validation
- `change_contact(args, contacts)`: Updates an existing contact with validation

**Error Handling:**
- Returns descriptive error messages for invalid inputs
- Prevents duplicate contact names
- Validates required arguments for each command

**Usage:**
```python
contacts = {}
add_contact(["John", "1234567890"], contacts)  # Returns 'Contact added.'
change_contact(["John", "9876543210"], contacts)  # Returns 'Contact updated.'
```

## Key Concepts

- **Caching/Memoization**: Storing function results to avoid redundant computation
- **Generators**: Using `yield` for memory-efficient iteration
- **Regular Expressions**: Pattern matching for text parsing
- **Decorators**: Wrapping functions to add error handling
- **Type Hints**: Using Python typing annotations for better code clarity
